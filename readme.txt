Glow360 美容服務預約平台

主要功能：
1. 首頁
   - 熱門商家展示
   - 新進商家推薦
   - 服務類別瀏覽
   - 客戶評價展示
   - 依地區搜尋服務

2. 商家列表頁面
   - 進階搜尋和篩選功能
   - 地圖/列表切換視圖
   - 分類標籤快速篩選
   - 商家卡片預覽

   - 地圖功能尚未實作

技術堆疊：
- 前端框架：React
- UI框架：Tailwind CSS
- 路由：React Router
- 打包工具：Vite
- 版本控制：Git

開發環境設置：
1. 安裝依賴：
   npm install

2. 啟動開發服務器：
   npm run dev

3. 建立生產版本：
   npm run build

注意事項：
- 本專案目前處於開發階段
- 所有數據均為模擬數據
- 建議在Node.js 14.0.0或以上版本運行
